# noise_functions
Noise functions with interactive parameters in javascript

List of noises:
- Bilinear
- Bicubic
- Perlin
- Voronoi
- Wood texture using a sine and displacement with a perlin noise
- diamond-square algorithm, simulating the fBm (fractional brownian motion)
